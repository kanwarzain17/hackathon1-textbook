# Quickstart: Module 2 Development

## Prerequisites
- Completed Module 1 (foundational knowledge).
- Docusaurus environment (Node.js/npm).

## Local Development
1. Navigate to `frontend_book/`.
2. Add content to `docs/modules/physical-ai/module-2-digital-twin/`.
3. Verify sidebar registration in `sidebars.ts`.
4. Run `npm start` for real-time preview.

## Success Criteria Validation
- **SC-004**: SCAN all new content for "LLM", "GPT", or "Training" to ensure no AI training content is included.
- **SC-001**: Verify both Gazebo and Unity roles are distinguished in the text.
